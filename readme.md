# これはなんですか？
GASを利用して、スプレッドシートから投稿画像＆キャプションを取得し、Instagramに投稿するツール  

# 何に使いますか
Instagramの自動投稿に利用します

# どう使いますか
↓の用な感じでスプレッドシートに記述して自動投稿を行っていました。
